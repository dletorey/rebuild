So I can write about what I'm [passionate](https://letorey.co.uk/about) about, you know:

The industry I work in changes daily, every morning I read about a new practice or JavaScript Library or whatever. So this will allow me to write about what I've learnt in the building of this humble site.

The development of this blog can be found on this [Github Repository](https://github.com/dletorey/letorey.co.uk).

This blog is built on https://matrix.squiz.net Squiz Matrix CMS, it's what I do during the week and probably way overkill for a simple blog, but will allow me to talk about development and new features in Squiz Matrix.
